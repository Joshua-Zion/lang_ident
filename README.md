# 🌎 FastText 多模型语种检测器

这是一个基于 Python Flask 后端和 HTML/Tailwind CSS 前端的实时语种检测 Web 应用。用户可以在网页端选择不同的 FastText 模型（包括自定义训练的模型和官方模型）来对输入的文本进行即时语言识别。

## ✨ 主要特性

- **多模型支持:** 可在前端自由切换三个预设的模型进行检测。
- **实时检测:** 通过 AJAX 请求实现文本输入后即时获取检测结果（点击“确定”按钮）。
- **中文映射:** 将 FastText 返回的 ISO 语言代码（如 `en`, `zh`）自动转换为对应的中文名称（如 `英语`, `中文 (汉语)`）。
- **美观界面:** 使用 Tailwind CSS 框架，确保了界面的美观性和响应式设计。

## ⚙️ 先决条件

在运行此应用之前，请确保你的系统已安装以下环境：

- **Python** (3.9 或更高版本)
- **pip** (Python 包管理器)

## 📦 设置和安装

1. ### 依赖安装

首先，安装项目所需的 Python 依赖：

```Plain
pip install flask fasttext
```

1. ### 文件结构准备

请确保你的项目目录结构如下，并将相应的模型文件放置到位：

```Plain
.
├── app.py              # Flask 后端核心逻辑
├── index.html          # Web 前端界面
└── model/
    ├── lid.176.bin     # 官方模型（如使用）
    └── lid.176.ftz     # 官方模型（如使用）
```

**重要提示：**

- `app.py` 和 `index.html` 必须在项目根目录。
- `lid.176.bin` 和 `lid.176.ftz` 模型文件必须放在名为 `model` 的子目录下。

## 🚀 使用指南

1. ### 启动应用

在项目根目录下运行 `app.py` 脚本以启动 Flask 后端服务：

```Plain
python app.py
```

终端将输出模型加载状态和服务器运行地址（通常是 `http://127.0.0.1:5000/`）。

```Plain
--- 正在加载 FastText 模型 ---
   ✅ 模型 'model/lid.176.bin' 加载成功。
   ✅ 文件 'model/lid.176.ftz' 加载成功。
 * Running on [http://127.0.0.1:5000/](http://127.0.0.1:5000/) (Press CTRL+C to quit)
```

1. ### 访问 Web 界面

打开浏览器，访问终端中显示的地址 (例如：`http://127.0.0.1:5000/`)。

1. ### 进行检测

1. 在 **"选择检测模型"** 下拉框中，选择你想要使用的 FastText 模型。
2. 在文本框中输入待检测的文本。
3. 点击 **"点击确定进行检测"** 按钮。

检测结果将实时显示在结果区域，包括语言的**中文名称**、**语言代码**以及**置信度**。

## 🛠️ 模型配置说明

所有模型路径配置都集中在 `app.py` 顶部的 `MODEL_PATHS` 字典中。

```Plain
MODEL_PATHS = {
    "lid.176.bin": "model/lid.176.bin", # 官方模型（位于 model/ 目录）
    "lid.176.ftz": "model/lid.176.ftz", # 官方模型（位于 model/ 目录）
}
```

如果你希望增加或更改模型，请：

1. 将新的模型文件放入项目目录中。
2. 更新 `MODEL_PATHS` 字典，添加新的键值对（键为前端显示的名称，值为文件路径）。
3. 同时，更新 `index.html` 中 `` 标签内的 `` 列表，以使新模型出现在前端下拉框中。
